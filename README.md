# Time Tracker

Projeto fullstack para controle de tempo de atividades.

## Tecnologias

* Frontend: React
* Backend: C# / ASP.NET Core

## Estrutura

time-tracker/
├── time-tracker (frontend)
└── time-tracker-api (backend)

## Como rodar

### Backend

cd time-tracker-api
dotnet run

### Frontend

cd time-tracker
npm install
npm run dev

## Funcionalidades

* Criar atividade
* Iniciar e parar timer
* Listar atividades
* Remover atividade

## Objetivo

Projeto criado para prática de desenvolvimento fullstack com C# e React.
